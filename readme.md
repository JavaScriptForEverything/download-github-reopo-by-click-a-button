This is a Full-Stack App | Nextjs-Express-Typescript project

## Getting Started

We need to run 2 server:
- Frontend Server 	: yarn dev      or      yarn start (build first)
- Backend Server 	: yarn dev      or      yarn start (build first)


```bash
yarn install

yarn dev
# or
yarn build 
yarn start
```
